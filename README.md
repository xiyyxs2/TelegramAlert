# Telegram Sim Alert Bot

Автопостер для Telegram-каналу у стилі живого адміна: тривоги, відбої, короткі оновлення, моніторинг, хвилі та повтори старих повідомлень.

## Важливо

Проект призначений для розважального/художнього каналу. Не використовуйте його як офіційне джерело реальних оповіщень.

У самі повідомлення бот НЕ додає фрази типу `симуляційний канал` або `не офіційне сповіщення`.

## Що вміє

- сам планує пости на день;
- сам постить у канал;
- робить тривоги й відбої;
- відбій збігається з регіоном активної тривоги;
- пише короткі оновлення у стилі каналу;
- іноді повторює свої попередні повідомлення;
- не додає донати;
- не постить картинки;
- не генерує великі полотна тексту.

## Railway Variables

```env
BOT_TOKEN=новий_токен_бота
ADMIN_ID=твій_telegram_id
CHANNEL_ID=-100xxxxxxxxxx
OPENWEATHER_API_KEY=
TIMEZONE=Europe/Kyiv
MIN_POSTS_PER_DAY=15
MAX_POSTS_PER_DAY=45
SIMULATION_LABEL=false
```

## Запуск локально

```bash
pip install -r requirements.txt
cp .env.example .env
python main.py
```

## Railway

`Procfile` вже готовий:

```text
worker: python main.py
```

## Блоговые сообщения

В этой версии есть отдельный блоговый режим. Он иногда пишет короткие посты в стиле живого админа, в том числе на разных языках мира: украинский, английский, арабский, испанский, французский, немецкий, японский, корейский и т.д.

В блоговые сообщения не добавлены донаты, картинки и реквизиты. Это просто случайные короткие посты для атмосферы канала.
